## 介绍
根据特定内容回复

## 安装
- 可以直接在astrbot的插件市场搜索astrbot_plugin_reply，点击安装

- 或者可以直接克隆源码到插件文件夹：

```bash
# 克隆仓库到插件目录
cd /AstrBot/data/plugins
git clone https://github.com/Anin-interest/astrbot_plugin_reply.git

# 控制台重启AstrBot
```

## 配置

在`resources/reply.yaml`中配置关键词和回复语
